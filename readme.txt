Calculadora MIUI Clone

Este é um projeto de uma calculadora clone da interface MIUI, desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

*   **Operações Básicas:** Adição (+), subtração (-), multiplicação (×) e divisão (÷).
*   **Operações Adicionais:** Porcentagem (%), troca de sinal (+/-), parênteses.
*   **Modo Científico:**
    *   Funções trigonométricas (sin, cos, tan).
    *   Logaritmos (log, ln).
    *   Potência (xʸ) e raiz quadrada (√).
    *   Constante Pi (π).
*   **Histórico:** Visualize e limpe o histórico de cálculos recentes.
*   **Gerador de Tabuada:** Gere a tabuada de qualquer número.
*   **Jogo "Enigma Numérico":** Um mini-jogo para adivinhar um número secreto entre 1 e 100.
*   **Design Responsivo:** A calculadora se adapta a diferentes tamanhos de tela.

## Como Usar

1.  Baixe os arquivos do projeto (`index.html`, `styles.css`, `script.js`).
2.  Abra o arquivo `index.html` em seu navegador de preferência.

## Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript
*   Phosphor Icons (para os ícones)
